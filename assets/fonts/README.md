# SF Pro fonts (iOS)

This folder contains the **SF Pro** font family used by the app on iOS.

## Configured in app (pubspec.yaml)

| File | Use |
|------|-----|
| `SF-Pro-Text-Regular.otf` | Body / default text |
| `SF-Pro-Text-Medium.otf` | Medium (500) |
| `SF-Pro-Text-Semibold.otf` | Semibold (600) |
| `SF-Pro-Text-Bold.otf` | Bold (700) |
